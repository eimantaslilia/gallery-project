    About the Project


  • This is a personal project to showcase CRUD operations using REST API on images.

  • The images are stored in a bucket using Amazon S3

  • The information about the images is stored in a MySQL database using Amazon RDS

  • Accounts have been setup using Spring Security with which the users can test the full functionality which includes uploading and deleting images
  • Built with:

  • Frontend: Thymeleaf, HTML, CSS, Bootstrap.

  • Backend: Spring Boot, Hibernate, MySQL, Maven.

  • More information can be found at the Github repository.
